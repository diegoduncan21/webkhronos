Registrarse

Esta sección está destinada para aquellas personas que aún no han utilizado nunca Khronos y desean crear un nuevo usuario.

El proceso para registrar un nuevo usuario consiste de cargar en la plantilla inicial un nombre de usuario, una dirección de correo electrónico, contraseña y verificación de contraseña.

El sistema verifica que todos los campos no se encuentren vacíos, además que el nombre de usuario no coincida con uno ya existente, que las contraseñas ingresadas sean coincidentes entre sí, entre otros. En caso de que en alguno de los campos falte información o no sea conincidente con los correspondientes, el sistema soicitará su re-ingreso.

Una vez verificado todo eso, el sistema registra al usuario y realiza un logueo automático para el mismo.

Logueo/Deslogueo

Aquí el usuario ya registrado debe ingresar su nombre de usuario y contraseña para empezar a utilizar Khronos. En caso de un ingreso incorrecto en alguno de los datos, el sistema solicitará su re-ingreso.

Una vez verificada la coincidencia de los datos registrados, el sistema deplegará al usuario el panel principal de Khronos.

Para el deslogueo, el usuario debe solo hacer click en su nombre de usuario que aparece en la parte baja del panel principal de Khronos. De esta manera, Khronos estará ahora disponible para que otro usuario se loguee o bien para el registro de uno nuevo.

Agregar Proyecto

Menú que permite agregar un nuevo proyecto. Para agregar un proyecto hacemos click en el signo más del módulo “Proyectos” y nos va a pedir que definamos un Nombre y luego pasamos a definir las tareas que lo componen.

Agregar Tarea

Se agregan y definen las tareas necesarias para llevar adelante el proyecto, se agrega una breve una descripción o anotaciones en cada una.

Seleccionar Proyecto

En una tabla ordenada podemos buscar entre una lista de proyectos existentes y creados por el usuario, alguno en particular. Simplemente se debe hacer click sobre el nombre de proyecto en el que se va a trabajar para ver las tareas que lo componen y su descripción.

Seleccionar Tarea

Una vez que se ha seleccionado un proyecto, el sistema ha desplegado la lista de tareas que lo componen. Para seleccionar una de ellas, el usuario debe nada más hacer click en la elegida.

Como resultado, el sistema mostrará la lista de intervalos que compnen dicha tarea.

En este punto, teniendo ya seleccionados un proyecto y una tarea, podemos empezar a utilizar el cronómetro.

Manejar Cronómetro

En esta sección el usuario puede empezar, finalmente, a utilizar el cronómetro. El mismo mostrará el avance en el tiempo que se encuentra transcurriendo para un determinado proyecto y para una determinada tarea.

Para comenzar, el usuario previamente ha seleccionado el proyecto sobre el que quiere trabajar, como así también la tarea.

El usuario debe hacer click sobre el botón "play" para empezar a contabilizar el tiempo. En este punto el usuario puede, o bien pausar el cronómetro, o bien detenerlo. Solo en el caso de que lo haya pausado podrá volver a poner "play", de esa manera el cronómetro seguirá corriendo.

Búsqueda de intervalos

Para conocer los intervalos de tiempo el usuario debe seleccionar un proyecto y luego una tarea. Una vez que se han seleccionado, el sistema muestra una lista ordenada de los mismos.

Ver estadísticas

Esta herramienta nos permite poder ver un gráfico representativo del avance del proyecto. Puede optar por dos alternativas, un diagrama circular u otro de barras.

Cambiar Ajustes

En este menú podemos Activar/Desactivar las descripciones que aparecen en cada botón (tooltips). También podemos encontrar las opciones para cambiar la contraseña del usuario.
